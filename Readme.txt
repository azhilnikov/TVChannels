Application shows a list of TV channels and TV programs of every channel.
List of TV channels shows a list of channels names.
List of TV programs shows program’s title, start date and time, program’s image and description.
The application written in Swift 3.1 and tested in Xcode 8.3.3.
Application supports iPhone and iPad with multiple device orientations.
Supported iOS versions: iOS 8.0 - iOS 10.3.2.
To open and build the project in XCode run TVChannelsTests.xcworkplace.
The app uses Gloss and Kingfisher frameworks to parse JSON objects and download images respectively.
